Detailed Description

The AIInsightsCredentials protocol is implemented by all classes that can provide identification of the application and keys for signing/encrypting data within the SDK.

Currently the Public Key of an Amazon Insights Identifier also acts as the Application key. All implementors must ensure they adhere to the NSCopying protocol in order to be able to use the AIInsightsCredential implementation as a key for caching of AImazonInsights instances in the SDK.
